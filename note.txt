Simple Mental Model
Layer	Think Like
Route	Receptionist
Service	Worker
Database	Storage
Schema	Security check
Model	Table structure

-------Topics------
relationships
foreign keys
authentication
JWT
Alembic migrations
PostgreSQL

MODEL - What table do I need?
SCHEMA - What request data should be allowed?, What response should return?
SERVICE - What logic should happen?
ROUTE - What endpoint should client call?


architecture - model, schema, service, router
