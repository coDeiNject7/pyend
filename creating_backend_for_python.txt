step 3 : inside pyend folder create venv(python3 -m venv $venv-name)
step 4 : activate venv (inside backend folder open terminal and run this command (source $venv-name/bin/activate)) and install required package's (uvicorn, fastapi, pydantic, sqlalchemy, alembic, redis, celery)

step 5 : write basic code in main.py

from fastapi import fastapi

app = FastAPI()

@app.get("/")
def root():
    return "Back-End is Running successfully !"
    
step 6 : run the project, open the terminal activate the venv run this command(uvicorn app.main:app --reload)

step 7 : open this http://127.0.0.1:8000 on the bowser we get this "Back-End is Running successfully !", and to view the swagger/documentation of api open this on browser http://127.0.0.1:8000/redoc or http://127.0.0.1:8000/docs
